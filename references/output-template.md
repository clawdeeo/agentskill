# AGENTS.md Output Template

## Structure

1. **Overview** — 1-2 sentence philosophy
2. **Cross-Language Patterns** — shared conventions (if any)
3. **Per-Language Sections** — one per detected language:
   - Naming (case, length, descriptiveness)
   - Error Handling
   - Comments
   - Spacing
4. **Git** — commits, branches
5. **Tooling** — global tools
6. **Red Lines** — explicit avoidances
7. **Footer** — sources + confidence

## Section Order

Overview → Cross-Language → Languages (alphabetical) → Git → Tooling → Red Lines → Footer

## Tone

Direct, imperative. Match commit message tone.
